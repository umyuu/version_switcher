# version_switcher
chrome 拡張。
Java言語のドキュメントのページにドキュメントバージョンを選択できるリストボックスを追加。
リストボックスの要素を選択すると選択したバージョンのページにリダイレクトする。
![chrome extension image](https://github.com/umyuu/version_switcher/blob/master/doc/images/version_switcher.PNG)
## インストール

## 使い方。
画像のリンクをドラック＆ドロップ。
